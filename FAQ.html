Question-1: What are the different ways to select an element in the DOM?

You can select an element in the DOM using getElementById(), getElementsByClassName(), or querySelector(). These methods help find elements by their ID, class, or CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets the HTML inside an element, including tags. innerText gets or sets the visible text only. textContent gets or sets all the text, even hidden ones, without tags.

Question-3: What is event delegation in the DOM?

Event delegation is when a parent element handles events for its child elements, improving performance and simplifying code.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event triggered on a child element propagates up to its parent elements in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement(). To add it, use appendChild() or insertBefore(). To remove it, use removeChild() or remove()
